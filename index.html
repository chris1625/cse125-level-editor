CSE 125 Level Editor

This is a super-basic (and very hacky) tile-based level editor. Source can be found here. Here are a few pointers to get started:
  • Left-click a tile below to select it
  • Left-click a tile on the map or press SPACE to change it to the selected tile
  • Right-click a tile on the map or press ESC to clear it
  • Middle-click a tile on the map or press R to rotate it
  • Dragging also works for left and right clicking
  • The "choose file" dialog will import a map file
